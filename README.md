# Residential Network Design & Deployment (v1.0)

## Overview
A reliable, scalable, and secure multi-floor residential network architecture designed for seamless coverage and future-ready traffic management.

## Network Topology
![Network Diagram](diagrams/v1.0_2025-12-08_003342.png)

## Core Infrastructure
* **Router:** ASUS RT-AX59U AX4200
* **Switch:** TP-Link TL-SG2210P Managed Switch
* **Access Points:** 2 x Ruijie RG-RAP2200(E) (Ceiling-mounted)
* **Cabling:** CAT-6 Full Copper
* **Clients:** Windows 11 Desktops, Canon LBP6030B Printer

## Key Features
* **Multi-Floor Coverage:** Seamless Wi-Fi via ceiling-mounted APs.
* **Stable Connectivity:** Wired backbone for workstations & printers.
* **Security:** Static IP assignments, IPS/IDS, Guest isolation.
* **Scalability:** VLAN-ready managed architecture.

## Future Roadmap
* Full VLAN segmentation (Guest / IoT / Office).
* QoS for critical work traffic.
* Centralised monitoring (Zabbix/PRTG) & UPS integration.

## Project Motivation
* **Solving Connectivity Issues:** Addressed existing Wi-Fi and cable stability problems to ensure uninterrupted work.
* **Need for Enterprise Reliability:** Implemented a professional-grade architecture to ensure high performance and reliability in a residential environment.
* **Scalability & Security Requirements:** Designed with future growth in mind, incorporating VLAN isolation and secure traffic management.
* **Professional Skill Application:** Leveraged knowledge of managed switching, network design, and security protocols to deliver a real-world, production-ready solution.
