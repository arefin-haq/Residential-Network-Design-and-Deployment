 
                    RESIDENTIAL NETWORK DESIGN & DEPLOYMENT

[ OVERVIEW ]
A reliable, scalable, and secure multi-floor residential network 
architecture designed for seamless coverage and future-ready 
traffic management.

[ NETWORK TOPOLOGY ]
Refer to: v1.0 2025-12-08 003342.png

[ CORE INFRASTRUCTURE ]
* Router       : ASUS RT-AX59U AX4200
* Switch       : TP-Link TL-SG2210P Managed Switch
* Access Points: 2 x Ruijie RG-RAP2200(E) (Ceiling-mounted)
* Cabling      : CAT-6 Full Copper
* Clients      : Windows 11 Desktops, Canon LBP6030B Printer

[ KEY FEATURES ]
* Multi-Floor Coverage : Seamless Wi-Fi via ceiling-mounted APs.
* Stable Connectivity  : Wired backbone for workstations & printers.
* Security             : Static IP assignments, IPS/IDS, Guest isolation.
* Scalability          : VLAN-ready managed architecture.

[ FUTURE ROADMAP ]
* Full VLAN segmentation (Guest / IoT / Office).
* QoS for critical work traffic.
* Centralised monitoring (Zabbix/PRTG) & UPS integration.
